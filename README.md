# 🧮 Java Swing Calculator

A modern-looking, user-friendly, and fast **Java Swing–based desktop calculator application**.

## ✨ Features

- ➕ Addition  
- ➖ Subtraction  
- ✖ Multiplication  
- ➗ Division  
- √ Square root  
- % Percentage calculation  
- +/- Positive / Negative toggle  
- Modern UI design  
- Error handling  

## 🛠 Technologies
 
- Java Swing  
- OOP (Object-Oriented Programming)
